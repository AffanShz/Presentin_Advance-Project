Web Based Presentation App  
Aplikasi berbasis web ini digunakan untuk menampilkan detail tutorial per langkah. Jadi 
diawali dengan user creator tutorial melakukan login ke aplikasi. Lalu terdapat halaman 
Manajemen master tutorial, dimana pada halaman ini user dapat melakukan CRUD dat a 
master tutorial.  
Master tutorial terdiri dari data judul, kode mata kuliah, url presentation, url finished, 
creator email, created at, updated at. Data mata kuliah bisa didapatkan dari webservice. URL 
presentation adalah alamat yang dapat diakses user umum untuk melihat tu torial master 
dan detailnya dengan status “show” (hasilnya mirip dengan yang ditampilkan pada halaman 
notion ini). URL finished adalah alamat yang dapat diakses user umum untuk melihat tutorial 
master dan semua detailnya baik dengan status “show” maupun “h ide” (tampilkan dalam 
bentuk PDF).  
Kemudian terdapat halaman Manajemen detail tutorial dimana pada halaman ini user dapat 
melakukan CRUD data detail tutorial. Detail tutorial berelasi dengan master tutorial, satu 
data master bisa memiliki banyak detail (One -to-Many). Detail tutorial bisa be rupa text, 
gambar, code, atau URL. Detail tutorial memiliki urutan (order) dan status (show/hide). Code 
ditampilkan menggunakan Syntax Highlighter.  
Detail tutorial terdiri dari data text, gambar, code, url, order, status, created at, updated at.  
Pada aplikasi ini juga terdapat layanan yang dapat menyediakan data master tutorial untuk 
sistem lain yang membutuhkan (webservice server).
